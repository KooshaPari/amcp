================================================================================
CONSOLIDATION STRATEGY
================================================================================

1. Single Source of Truth
   Every fixture defined EXACTLY ONCE in appropriate module

2. Domain-Specific Fixtures Stay Local
   E2E, performance, production load fixtures remain in their conftest.py

3. Graceful Degradation
   Optional dependencies handled with try/except

4. Clear Ownership
   Each module documented with purpose and fixture list

5. Automatic Discovery
   pytest_plugins = ["tests.fixtures"] auto-loads all fixtures

PYTEST MARKERS (NEW)
================================================================================

@pytest.mark.smoke
  - Fast sanity checks (<1 second)
  - Run with: pytest -m smoke

@pytest.mark.slow
  - Long-running tests (>10 seconds)
  - Skip with: pytest -m "not slow"

@pytest.mark.integration
  - Tests requiring external services
  - Run with: pytest -m integration

@pytest.mark.performance
  - Performance and load tests
  - Run with: pytest -m performance
